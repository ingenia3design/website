Build a production-ready **single-page application** for **Ingenia3Design**, a company specialized in **3D printing, rapid prototyping, CAD modeling, and custom manufacturing**.

## Objective

Create a modern, premium, responsive website that works flawlessly on **desktop, tablet, and mobile devices**. The design should feel **high-tech, clean, innovative, and conversion-focused**. Prioritize performance, maintainability, and polished UI/UX.

## Preferred stack

Use the best stack for a responsive modern SPA. Preferred choice:

* **React + Vite + TypeScript**
* **Tailwind CSS**
* **Framer Motion** for subtle animations
* **React Icons** or **Lucide React**
* Optional: **Ionic React** only if it clearly improves mobile behavior, otherwise prefer a cleaner React/Tailwind solution

## Implementation requirements

* Use **clean architecture** with reusable components
* Use **functional React components**
* Use **TypeScript**
* Keep code modular and easy to scale
* Organize files into folders such as:

  * `components`
  * `sections`
  * `data`
  * `assets`
  * `hooks`
* Create a polished **mobile-first** layout
* Use a **sticky header**
* Add **smooth scrolling** navigation
* Add a **hamburger menu** for small screens
* Ensure **touch-friendly spacing and buttons**
* Keep the app lightweight and performant
* Use semantic HTML and accessibility best practices
