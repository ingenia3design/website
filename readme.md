# Ingenia3Design — SPA
version 1.0.1
Single-page app (React + Vite + TypeScript) para **Ingenia3Design** con **Tailwind CSS** y animaciones sutiles con **Framer Motion**.

## Requisitos

- Node.js 18+ (recomendado 20+)

## Ejecutar en local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
